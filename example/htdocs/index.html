PassBy[ME] PHP client Messaging Examples

This example demonstrate, how PassBy[ME] works with the passbyme2fa-client PHP application.

First of all read the Get Started and configuration sections from the project GitHub page. Create your own config.php file under "conf" folder by making a copy of the config.dist.php file and set your own configuration.

If you use PassBy[ME] the first time you need a registered user with an enrolled device to send messages to. Users can be created in two ways:

from an administrator

  1. Log in to your registered account (organization).

  2. Click on the icon under "Administrators" menu.

  3. Add your users PassBy[ME] Id and press the "OK" button.

At the end of this process will you have a registered user in your PassBy[ME] system with the same enrolled device as your administrator.

individually

  1. Log in to your registered account (organization).

  2. Create a new user under "PassBy[ME] Users" menu by clicking on the "New user" button.

  3. Add a new device to your user by clicking on the icon.

  4. Click on "Add device enrollment" then download the enrollment sheet by clicking on the icon.

  5. Follow the instructions written on the enrollment sheet.

At the end of these steps you will have a registered user with an enrolled device.
