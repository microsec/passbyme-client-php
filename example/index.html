PassBy[ME] two factor authentication client code example

This example demonstrate, how PassBy[ME] authentication works with the passbyme2fa-client application.

First of all you have to configure the example properly to get it work. You can do that by creating a copy of the config.dist.php file and create your own config.php.

For more details about the configuration properties please check the README.md file.
